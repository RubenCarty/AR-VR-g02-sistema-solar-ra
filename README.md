## Instalación y Uso

### Requisitos

- Unity 2022.3.62f1 (LTS)
- Android 7.0+ (API 24+) con ARCore compatible
- AR Foundation 5.1.x
- AR Foundation 5.x + ARCore XR Plugin

### Clonar el repositorio

```bash
git clone https://github.com/RubenCarty/rva-g02-sistema-solar-ra.git
cd rva-g02-sistema-solar-ra
```

### Abrir en Unity

1. Abrir Unity Hub
2. "Add" → seleccionar la carpeta del proyecto
3. Abrir con Unity 2022.3.62f1
4. `File → Build Settings → Android → Build and Run`
